Guía Rápida de Comandos

Comandos útiles para terminal en Windows y Linux.

Windows

  • systeminfo — Ver información del sistema
  • tasklist — Listar procesos activos
  • shutdown /s /t 0 — Apagar el sistema inmediatamente
  • ping [ip] — Verificar conectividad con una IP
  • ipconfig /all — Ver configuración IP completa
  • tracert [dominio] — Ver ruta hasta un destino
  • nslookup [dominio] — Ver resolución DNS
  • net use — Ver conexiones de red
  • color xy — cambiar color de la consola
  • net localgroup administrators — Ver administradores

Linux

  • uptime — Ver tiempo encendido del sistema
  • free -h — Ver memoria disponible
  • lsblk — Ver dispositivos de bloques
  • mount — Ver dispositivos montados
  • uname -a — Ver información del sistema
  • ps aux — Ver todos los procesos
  • whoami — Mostrar usuario actual
  • chmod +x script.sh — Dar permisos de ejecución
  • chown user:group archivo — Cambiar propietario y grupo
  • journalctl -xe — Ver logs del sistema
  • ifconfig — Ver configuración de red
  • df -h — Ver espacio en disco
  • top — Monitor de procesos

GitHub

  • git clone URL — Clonar un repositorio
  • git log — Ver historial de commits
  • git branch — Ver ramas locales
  • git checkout -b nueva-rama — Crear y cambiar a nueva rama
  • git add . — Agregar todos los archivos al staging
  • git commit -m "mensaje" — Crear un commit
  • git push origin main — Subir cambios al repositorio remoto
  • gh repo create — Crear un repositorio nuevo
  • gh pr create — Crear Pull Request
  • gh issue list — Listar issues abiertas
  • git init — Inicializar repositorio
  • git status — Ver estado
  • gh auth login — Iniciar sesión con GitHub CLI

macOS

  • defaults write com.apple.finder AppleShowAllFiles YES — Mostrar archivos ocultos
  • killall Finder — Reiniciar Finder
  • brew install nombre — Instalar app con Homebrew
  • top — Ver procesos activos
  • df -h — Ver uso del disco
  • say "Hola mundo" — Leer texto con voz
  • open . — Abrir carpeta actual en Finder
  • sw_vers — Ver versión del sistema

Fortinet

  • config system global — Entrar a configuración global
  • set admin-sport 444 — Cambiar puerto del admin GUI
  • end — Salir de modo configuración
  • show full-configuration — Ver configuración completa
  • diag debug config-error-log read — Leer errores de configuración
  • show — Mostrar configuración actual
  • diagnose debug enable — Habilitar modo debug
  • execute backup config flash — Hacer backup de la configuración
  • get system status — Ver estado del sistema
