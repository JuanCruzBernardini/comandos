🧠 Guía Rápida de Comandos

Comandos útiles para terminal en Windows y Linux.

Windows

  • 🌐 Red:
  • ipconfig /all — Ver configuración IP completa
  • arp -a — Ver tabla ARP
  • tracert [dominio] — Ver ruta hasta un destino
  • nslookup [dominio] — Ver resolución DNS
  • net use — Ver conexiones de red
  • 👥 Usuarios:
  • net user — Ver usuarios del sistema
  • net localgroup administrators — Ver administradores
  • 🛡️ Seguridad:
  • gpresult /R — Ver políticas de grupo aplicadas
  • wmic useraccount get name,sid — Ver SID de usuarios
  • cipher /w:C: — Borrar espacio libre en disco
  • ver — Ver versión del sistema operativo
  • assoc — Ver asociaciones de extensiones
  • tree — Ver estructura de carpetas
  • net config workstation — Ver configuración de red
  • wmic logicaldisk get caption,description — Ver discos

Linux

  • 🌐 Red:

    • Linux

    • ip r — Ver tabla de rutas
    • dig [dominio] — Consultas DNS
    • traceroute [dominio] — Ver ruta hasta destino
    • 👥 Usuarios:
    • cat /etc/passwd — Ver usuarios del sistema
    • cat /etc/shadow — Ver hashes de contraseñas (requiere root)
    • groups — Ver grupos del usuario actual
    • id usuario — Ver UID, GID y grupos
    • 🔐 Permisos y propiedad:
    • chmod 755 archivo — Cambiar permisos
    • chown usuario:grupo archivo — Cambiar propietario
    • 🔎 Búsqueda y archivos:
    • find / -name archivo — Buscar archivos
    • locate archivo — Buscar rápido (requiere updatedb)
    • grep 'texto' archivo — Buscar texto en archivo
    • tail -f archivo — Ver últimas líneas en tiempo real
    • head archivo — Ver primeras líneas
    • alias — Ver o crear alias de comandos
    • lsblk — Ver dispositivos de bloques
    • mount / umount — Montar/desmontar unidades
    • 🧰 Herramientas:
    • nc -lvnp [puerto] — Escuchar conexiones (netcat)
    • wget URL — Descargar desde web
    • curl URL — Hacer peticiones HTTP
    • service [servicio] start|stop|restart — Administrar servicios
    • systemctl status [servicio] — Ver estado de servicios
    • who — Ver usuarios logueados
    • last — Ver historial de logins
    • ls -lah — Listar archivos con permisos y tamaño legible
    • ncdu — Ver uso de espacio (si está instalado)
